## Content

VkAccelerationStructureNV - Opaque handle to an acceleration structure object

Acceleration structures for the `[VK_NV_ray_tracing](VK_NV_ray_tracing.html) extension` are
represented by the similar `VkAccelerationStructureNV` handles:

// Provided by VK_NV_ray_tracing
VK_DEFINE_NON_DISPATCHABLE_HANDLE(VkAccelerationStructureNV)
