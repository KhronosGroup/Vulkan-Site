VkHeadlessSurfaceCreateFlagsEXT - Reserved for future use

// Provided by VK_EXT_headless_surface
typedef VkFlags VkHeadlessSurfaceCreateFlagsEXT;

`VkHeadlessSurfaceCreateFlagsEXT` is a bitmask type for setting a mask,
but is currently reserved for future use.

[VK_EXT_headless_surface](VK_EXT_headless_surface.html), `VkFlags`, [VkHeadlessSurfaceCreateInfoEXT](VkHeadlessSurfaceCreateInfoEXT.html)
