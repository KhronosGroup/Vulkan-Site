VKAPI_ATTR - Vulkan function attribute macro

`VKAPI_ATTR` is a macro placed before the return type in Vulkan API
function declarations.
This macro controls calling conventions for C++11 and GCC/Clang-style
compilers.

[VKAPI_CALL](VKAPI_CALL.html), [VKAPI_PTR](VKAPI_PTR.html)

For more information, see the [Vulkan Specification](../../../../spec/latest/appendices/boilerplate.html#boilerplate-platform-specific-calling-conventions).
