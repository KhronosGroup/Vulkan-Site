// Provided by VK_VERSION_1_0
typedef VkFlags VkPipelineCreateFlags;

`VkPipelineCreateFlags` is a bitmask type for setting a mask of zero or
more [VkPipelineCreateFlagBits](VkPipelineCreateFlagBits.html).
