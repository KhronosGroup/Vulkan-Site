VkGpaPerfBlockPropertiesFlagsAMD - Reserved for future use

// Provided by VK_AMD_gpa_interface
typedef VkFlags VkGpaPerfBlockPropertiesFlagsAMD;

`VkGpaPerfBlockPropertiesFlagsAMD` is a bitmask type for setting GPA
performance block property flags, but is currently reserved for future use.
