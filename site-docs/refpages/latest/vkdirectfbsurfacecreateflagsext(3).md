VkDirectFBSurfaceCreateFlagsEXT - Reserved for future use

// Provided by VK_EXT_directfb_surface
typedef VkFlags VkDirectFBSurfaceCreateFlagsEXT;

`VkDirectFBSurfaceCreateFlagsEXT` is a bitmask type for setting a mask,
but is currently reserved for future use.

[VK_EXT_directfb_surface](VK_EXT_directfb_surface.html), [VkDirectFBSurfaceCreateInfoEXT](VkDirectFBSurfaceCreateInfoEXT.html), `VkFlags`
