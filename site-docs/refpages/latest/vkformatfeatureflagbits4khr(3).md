VkFormatFeatureFlagBits4KHR - Bitmask specifying features supported by a buffer

Bits which **can** be set in the [VkFormatProperties4KHR](VkFormatProperties4KHR.html) features
`linearTilingFeatures`, `optimalTilingFeatures`, and
`bufferFeatures` are:

// Provided by VK_KHR_extended_flags
// Flag bits for VkFormatFeatureFlagBits4KHR
typedef VkFlags64 VkFormatFeatureFlagBits4KHR;

[VK_KHR_extended_flags](VK_KHR_extended_flags.html), [VkFormatFeatureFlags4KHR](VkFormatFeatureFlags4KHR.html)

For more information, see the [Vulkan Specification](../../../../spec/latest/chapters/formats.html#VkFormatFeatureFlagBits4KHR).
