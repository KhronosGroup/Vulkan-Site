VkCommandBufferUsageFlags - Bitmask of VkCommandBufferUsageFlagBits

// Provided by VK_VERSION_1_0
typedef VkFlags VkCommandBufferUsageFlags;

`VkCommandBufferUsageFlags` is a bitmask type for setting a mask of zero
or more [VkCommandBufferUsageFlagBits](VkCommandBufferUsageFlagBits.html).

[VK_VERSION_1_0](VK_VERSION_1_0.html), [VkCommandBufferBeginInfo](VkCommandBufferBeginInfo.html), [VkCommandBufferUsageFlagBits](VkCommandBufferUsageFlagBits.html), `VkFlags`
