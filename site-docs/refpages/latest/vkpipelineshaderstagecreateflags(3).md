VkPipelineShaderStageCreateFlags - Bitmask of VkPipelineShaderStageCreateFlagBits

// Provided by VK_VERSION_1_0
typedef VkFlags VkPipelineShaderStageCreateFlags;

`VkPipelineShaderStageCreateFlags` is a bitmask type for setting a mask
of zero or more [VkPipelineShaderStageCreateFlagBits](VkPipelineShaderStageCreateFlagBits.html).
