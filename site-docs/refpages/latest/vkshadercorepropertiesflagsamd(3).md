VkShaderCorePropertiesFlagsAMD - Bitmask of VkShaderCorePropertiesFlagBitsAMD

// Provided by VK_AMD_shader_core_properties2
typedef VkFlags VkShaderCorePropertiesFlagsAMD;

`VkShaderCorePropertiesFlagsAMD` is a bitmask type for providing zero or
more [VkShaderCorePropertiesFlagBitsAMD](VkShaderCorePropertiesFlagBitsAMD.html).
