## Content

ANativeWindow - Android native window type

To remove an unnecessary compile time dependency, an incomplete type
definition of `ANativeWindow` is provided in the Vulkan headers:

// Provided by VK_KHR_android_surface
struct ANativeWindow;

The actual `ANativeWindow` type is defined in Android NDK headers.

[VK_KHR_android_surface](VK_KHR_android_surface.html)

For more information, see the [Vulkan Specification](../../../../spec/latest/chapters/VK_KHR_surface/wsi.html#ANativeWindow).

This page is extracted from the Vulkan Specification.
Fixes and changes should be made to the Specification, not directly.
