VkUbmSurfaceCreateFlagsSEC - reserved flag for surface creation

// Provided by VK_SEC_ubm_surface
typedef VkFlags VkUbmSurfaceCreateFlagsSEC;

`VkUbmSurfaceCreateFlagsSEC` is a bitmask type for setting a mask, but
is currently reserved for future use.

[VK_SEC_ubm_surface](VK_SEC_ubm_surface.html), `VkFlags`, [VkUbmSurfaceCreateInfoSEC](VkUbmSurfaceCreateInfoSEC.html)
