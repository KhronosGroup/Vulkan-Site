VkSurfaceCreateFlagsOHOS - reserved flag for surface creation

// Provided by VK_OHOS_surface
typedef VkFlags VkSurfaceCreateFlagsOHOS;

`VkSurfaceCreateFlagsOHOS` is a bitmask type for setting a mask, but is
currently reserved for future use.

[VK_OHOS_surface](VK_OHOS_surface.html), `VkFlags`, [VkSurfaceCreateInfoOHOS](VkSurfaceCreateInfoOHOS.html)
