VkDeviceFaultFlagsKHR - Bitmask specifying valid fields of a fault info structure

// Provided by VK_KHR_device_fault
typedef VkFlags VkDeviceFaultFlagsKHR;

`VkDeviceFaultFlagsKHR` is a bitmask type for receiving a mask of zero
or more [VkDeviceFaultFlagBitsKHR](VkDeviceFaultFlagBitsKHR.html).

[VK_KHR_device_fault](VK_KHR_device_fault.html), [VkDeviceFaultFlagBitsKHR](VkDeviceFaultFlagBitsKHR.html), [VkDeviceFaultInfoKHR](VkDeviceFaultInfoKHR.html), `VkFlags`
