VkCommandPoolCreateFlags - Bitmask of VkCommandPoolCreateFlagBits

// Provided by VK_VERSION_1_0
typedef VkFlags VkCommandPoolCreateFlags;

`VkCommandPoolCreateFlags` is a bitmask type for setting a mask of zero
or more [VkCommandPoolCreateFlagBits](VkCommandPoolCreateFlagBits.html).

[VK_VERSION_1_0](VK_VERSION_1_0.html), [VkCommandPoolCreateFlagBits](VkCommandPoolCreateFlagBits.html), [VkCommandPoolCreateInfo](VkCommandPoolCreateInfo.html), `VkFlags`
