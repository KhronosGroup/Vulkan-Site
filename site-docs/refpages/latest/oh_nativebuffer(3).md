OH_NativeBuffer - Native buffer on Open Harmony OS platform

The `OH_NativeBuffer` structure is defined as:

// Provided by VK_OHOS_external_memory
struct OH_NativeBuffer;

It is the native buffer structure on Open Harmony OS platform.
It is defined in Open Harmony OS NDK headers.

[VK_OHOS_external_memory](VK_OHOS_external_memory.html)

For more information, see the [Vulkan Specification](../../../../spec/latest/chapters/resources.html#OH_NativeBuffer).
