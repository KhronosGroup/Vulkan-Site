VkVideoEncodeContentFlagsKHR - Bitmask specifying the video encode content flags

// Provided by VK_KHR_video_encode_queue
typedef VkFlags VkVideoEncodeContentFlagsKHR;

`VkVideoEncodeContentFlagsKHR` is a bitmask type for setting a mask of
zero or more [VkVideoEncodeContentFlagBitsKHR](VkVideoEncodeContentFlagBitsKHR.html).

[VK_KHR_video_encode_queue](VK_KHR_video_encode_queue.html), `VkFlags`, [VkVideoEncodeContentFlagBitsKHR](VkVideoEncodeContentFlagBitsKHR.html), [VkVideoEncodeUsageInfoKHR](VkVideoEncodeUsageInfoKHR.html)
