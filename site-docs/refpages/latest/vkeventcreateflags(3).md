VkEventCreateFlags - Bitmask of event creation flag bits

// Provided by VK_VERSION_1_0
typedef VkFlags VkEventCreateFlags;

`VkEventCreateFlags` is a bitmask type for setting a mask of
[VkEventCreateFlagBits](VkEventCreateFlagBits.html).

[VK_VERSION_1_0](VK_VERSION_1_0.html), [VkEventCreateFlagBits](VkEventCreateFlagBits.html), [VkEventCreateInfo](VkEventCreateInfo.html), `VkFlags`
