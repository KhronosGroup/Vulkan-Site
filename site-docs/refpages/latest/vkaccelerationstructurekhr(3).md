VkAccelerationStructureKHR - Opaque handle to an acceleration structure object

Acceleration structures for the `[VK_KHR_acceleration_structure](VK_KHR_acceleration_structure.html)
extension` are represented by `VkAccelerationStructureKHR` handles:

// Provided by VK_KHR_acceleration_structure
VK_DEFINE_NON_DISPATCHABLE_HANDLE(VkAccelerationStructureKHR)
