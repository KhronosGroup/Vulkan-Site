VkMemoryPropertyFlags - Bitmask of VkMemoryPropertyFlagBits

// Provided by VK_VERSION_1_0
typedef VkFlags VkMemoryPropertyFlags;

`VkMemoryPropertyFlags` is a bitmask type for setting a mask of zero or
more [VkMemoryPropertyFlagBits](VkMemoryPropertyFlagBits.html).

[VK_VERSION_1_0](VK_VERSION_1_0.html), `VkFlags`, [VkMemoryPropertyFlagBits](VkMemoryPropertyFlagBits.html), [VkMemoryType](VkMemoryType.html)

For more information, see the [Vulkan Specification](../../../../spec/latest/chapters/memory.html#VkMemoryPropertyFlags).
