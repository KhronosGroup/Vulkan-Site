# OpenGL Shading Language Extensions

## Metadata

- **Component**: glslext
- **Version**: latest
- **URL**: /glslext/latest/glslext/index.html

## Content

This component includes specifications of GLSL extensions, some of which are
needed when using GLSL with Vulkan.

GLSL extensions are written in an older style, as edits to specific versions
of the GLSL Specification, rather than incorporated in the GLSL
Specification.
The extension documents are not formatted with asciidoc (or any other)
markup, but are just text files.
