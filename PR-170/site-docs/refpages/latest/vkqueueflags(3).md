VkQueueFlags - Bitmask of VkQueueFlagBits

// Provided by VK_VERSION_1_0
typedef VkFlags VkQueueFlags;

`VkQueueFlags` is a bitmask type for setting a mask of zero or more
[VkQueueFlagBits](VkQueueFlagBits.html).
